# 道体·玄盾 部署指南（v1.2.0）

> **面向对象**：企业IT管理员、运维人员
> **适用版本**：v1.2.0
> **最后更新**：2026年7月

## 一、系统要求

### 桌面端

| 平台 | 最低配置 | 推荐配置 | 备注 |
|------|---------|---------|------|
| Windows | Win10 1903+ / 4GB RAM / 200MB | Win11 / 8GB RAM / SSD | 需 .NET Framework 4.7.2+ |
| macOS | macOS 12+ (Apple Silicon) / 4GB RAM | macOS 14+ / 8GB RAM | 仅支持 Apple Silicon |
| Linux | Ubuntu 20.04+ / 4GB RAM / 200MB | Ubuntu 22.04+ / 8GB RAM | 需 libwebkit2gtk-4.1 |

### SDK集成

| 项目 | 要求 |
|------|------|
| Python | 3.10+ |
| 依赖 | numpy, flask（自动安装） |

## 二、安装步骤

### 2.1 Windows安装

**标准安装**：
1. 下载 `XuanDun_1.2.0_x64-setup.exe`
2. 双击运行安装程序
3. 按向导选择安装路径
4. 完成安装后从开始菜单启动

**静默安装（企业批量部署）**：
```cmd
XuanDun_1.2.0_x64-setup.exe /S
```

### 2.2 macOS安装

1. 下载 `XuanDun_1.2.0_aarch64.dmg`
2. 打开DMG，拖拽到Applications
3. 首次打开：系统设置 → 隐私与安全性 → 点击"仍要打开"

### 2.3 Linux安装

**AppImage（便携版）**：
```bash
chmod +x XuanDun_1.2.0_amd64.AppImage
./XuanDun_1.2.0_amd64.AppImage
```

**deb包**：
```bash
sudo dpkg -i XuanDun_1.2.0_amd64.deb
```

### 2.4 SDK安装

```bash
pip install daoti-xuandun==1.2.0
```

## 三、首次启动验证

### 3.1 桌面端首次启动

1. 启动应用，配置向导自动出现
2. 选择防护模式（推荐"平衡模式"）
3. 向导完成后系统自动进入**观察模式**
4. 顶部状态条显示"🟡 观察模式（学习中）"

### 3.2 运行验证脚本

```bash
python scripts/verify_installation.py
```

脚本验证4个环节：环境检查 → 引擎连通性 → 基础防护测试 → 学习状态检查。全部通过即表示安装成功。

## 四、企业批量部署

### 4.1 标准化配置

1. 在测试机器上完成安装和配置向导
2. 调整设置后导出配置文件
3. 配置文件位置：
   - Windows: `%APPDATA%\com.daoti.xuandun-desktop\config.json`
   - macOS: `~/Library/Application Support/com.daoti.xuandun-desktop/config.json`
   - Linux: `~/.config/com.daoti.xuandun-desktop/config.json`

### 4.2 批量部署脚本

**Windows（PowerShell）**：
```powershell
$computers = @("PC01", "PC02", "PC03")
foreach ($pc in $computers) {
    Copy-Item "\\fileserver\share\XuanDun_1.2.0_x64-setup.exe" "\\$pc\c$\temp\"
    Invoke-Command -ComputerName $pc -ScriptBlock {
        Start-Process "C:\temp\XuanDun_1.2.0_x64-setup.exe" -ArgumentList "/S" -Wait
        Copy-Item "\\fileserver\share\config.json" "$env:APPDATA\com.daoti.xuandun-desktop\"
    }
}
```

**Linux（Bash）**：
```bash
#!/bin/bash
HOSTS=("server01" "server02" "server03")
for host in "${HOSTS[@]}"; do
    scp XuanDun_1.2.0_amd64.deb $host:/tmp/
    ssh $host "sudo dpkg -i /tmp/XuanDun_1.2.0_amd64.deb"
    scp config.json $host:~/.config/com.daoti.xuandun-desktop/
done
```

### 4.3 部署后验证

每台机器运行验证脚本：
```bash
python scripts/verify_installation.py --port 18765
```

## 五、常见问题排查

### Q1: 安装后应用无法启动

**Windows**：检查 .NET Framework ≥ 4.7.2，以管理员身份运行
**macOS**：`xattr -cr /Applications/道体·玄盾.app`
**Linux**：`sudo apt install libwebkit2gtk-4.1 libgtk-3-0`

### Q2: 引擎未启动（端口18765不可达）

```bash
# 检查引擎进程
tasklist | findstr xuandun-engine    # Windows
ps aux | grep xuandun-engine         # Linux/macOS
```

引擎进程不存在时，重启桌面端应用。

### Q3: 观察模式长时间不切换

- 默认需要1000条样本
- 可在设置中调低阈值
- 或手动切换：设置页 → 切换到保护模式

### Q4: 防护测试失败

- 确认当前为"保护模式"（观察模式不拦截）
- 运行模拟测试查看详细报告

## 六、升级与卸载

### 升级

**桌面端**：新版本覆盖安装，配置和数据自动保留
**SDK**：`pip install --upgrade daoti-xuandun`

### 卸载

**Windows**：控制面板 → 程序和功能 → 卸载
**macOS**：删除 Applications 中的应用
**Linux**：`sudo dpkg -r xuandun-desktop`

### 配置回滚

每次配置变更自动创建快照。回滚：设置页 → 配置历史 → 选择版本 → 回滚
