### S3：模块二 - 动态阴阳壳

**前置：S1。对应规约 §4。**

| 方法 | 规约章节 | 功能 |
|:--|:--|:--|
| `__init__` | §4.3 | 根据配置初始化所有权重矩阵（调用 `_init_weights`） |
| `_init_weights` | §4.4 | 用 `shell_key` 种子生成 `W_f, U_f, b_f, W_b, U_b, b_b` |
| `transform` | §4.3 | 执行 `t_iter` 次双向递归，输出混淆后向量 |

**双向递归细节（§4.1）**：
```
初始化 h_f[0] = x, h_b[num_layers+1] = zeros
For t in range(t_iter):
    For i in range(1, num_layers+1):
        h_f[i] = tanh(W_f @ h_f[i-1] + U_f @ h_b[i] + b_f)
    For i in reversed(range(1, num_layers+1)):
        h_b[i] = tanh(W_b @ h_b[i+1] + U_b @ h_f[i] + b_b)
输出 y = h_b[1]
```

**约束**：纯 `numpy` 实现，不依赖 `torch`。使用 `np.tanh` 确保数值稳定性。

**测试文件**：`tests/test_dynamic_shell.py`，包含：
- `test_shell_deterministic` — 相同输入产生相同输出
- `test_shell_sensitivity` — 不同密钥产生完全不同输出

### S4：模块三 - 古文符号映射

**前置：S1。对应规约 §5。**

| 方法 | 规约章节 | 功能 |
|:--|:--|:--|
| `__init__` | §5.2 | 用 `mapping_key` 种子生成随机置换表 `perm` |
| `map` | §5.2 | `vec` 每个分量 [-1,1] → 置换表索引 → 符号序列 |

**映射公式（§5.1）**：
```
idx[i] = floor((vec[i] + 1) / 2 * symbol_table_size)
clipped to [0, symbol_table_size - 1]
symbols[i] = perm[idx[i]]
```

### S5：模块四 - 时序一致性校验

**前置：S1。对应规约 §6。**

| 方法 | 规约章节 | 功能 |
|:--|:--|:--|
| `__init__` | §6.2 | 初始化窗口大小、阈值、会话状态字典 |
| `_extract_feature` | §6.2 | 符号序列 → 归一化直方图特征 |
| `check` | §6.2 | 计算马氏距离 → 滑动窗口管理 → `TimingDecision` |
| `_mahalanobis` | §6.3 | `sqrt((x-μ)^T · Σ⁻¹ · (x-μ))` |

**马氏距离实现细节**：
- 使用 `np.linalg.pinv` 求协方差矩阵的伪逆，防止奇异
- 窗口未满时直接放行
- 状态按 `session_id` 隔离，每个会话维护独立 FIFO 队列

**测试文件**：`tests/test_timing_checker.py`，包含：
- `test_timing_check_accept_repetitive` — 重复模式持续放行
- `test_timing_check_reject_anomaly` — 异常序列触发告警

### S6：主集成与端到端验证

**前置：S2-S5。对应规约 §7。**

| 产出文件 | 说明 |
|:--|:--|
| `xuandun.py` | 完善 `XuanDun` 类的 `protect()` 流水线方法 |
| `tests/test_xuandun.py` | 端到端集成测试 |
| `demo.py` | 展示完整防护流程，包含模拟攻击用例 |

**`protect()` 流水线逻辑（§7.1）**：
```
1. 若启用门禁 → gate.process(raw_input)
   └─ REJECT → 立即返回 ProtectResult(allowed=False)
2. 若启用阴阳壳 → shell.transform(vector)
3. 若启用符号映射 → mapper.map(obfuscated_vector)
4. 若启用时序校验 → checker.check(symbol_seq, session_id)
   └─ REJECT/WARN → 返回对应结果
5. 全通过 → ProtectResult(allowed=True)
```

**demo.py 场景要求**：
- 正常输入通过全流程
- 未知域输入被门禁拒绝
- 异常时序输入被一致性校验捕获

## 5. 风险与缓解

| 风险 | 缓解措施 |
|:--|:--|
| 马氏距离协方差奇异 | 使用 `np.linalg.pinv` + 小正则化单位矩阵 |
| 域分类器未提供 | 内置 `LightweightDomainClassifier` 作为后备 |
| 大维度下递归性能 | `hidden_dim` 默认 128，迭代次数默认 5，均在可控范围 |
| 数值溢出 | clip 操作 + float32 精度 |
